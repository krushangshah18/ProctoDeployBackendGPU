Timing Breakdown — Where Does Each Tick Go?

YOLO (GPU) and MediaPipe (CPU) run concurrently. Wall-clock concurrent cost = max(YOLO, MP). Overhead = tick_avg − max(YOLO, MP) — includes frame snapshot, state update, SSE push, asyncio scheduling. If overhead is large, the bottleneck is not inference.
